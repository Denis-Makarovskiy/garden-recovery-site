## Точные спеки (диффы состояний)

### Desktop-64 — «Путь гостя» (pinned scroll-scrub, 5 шагов)
- Аккордеон из 5 этапов (Консультация → Диагностика → Протокол → Интенсив → Стабилизация).
- На каждом шаге: активная панель раскрывается (высота `42 → 152..177px`), предыдущая схлопывается (`152 → 42`), нижние элементы переезжают вниз (`+25 / +51 / −110` по шагам — reflow).
- **Фото справа кроссфейдится** под активный этап (s1 = врач+гостья, s5 = руки с блокнотом).
- Реализация: ScrollTrigger `pin` секции + `scrub`; индекс активного шага = f(progress); высоты панелей и opacity фото — твины EASE_OUT.

### Desktop-66 — «Почему это работает» (scroll-scrub, 6 шагов = 4 карточки)
- Старт: только заголовок + лид. Затем 4 карточки появляются **по одной**.
- Каждая карточка: `x: +1430 → 0` (въезд справа), `scale: 284→307 ≈ 1.08`, opacity 0→1.
- Stagger между карточками; EASE_OUT.
- Реализация: ScrollTrigger на секцию, `gsap.from(cards, {xPercent:120, scale:0.92, autoAlpha:0, stagger, ease:'power3.out'})` (scrub или once).

### Desktop-50 — зум-секвенция изображений (8 шагов)
