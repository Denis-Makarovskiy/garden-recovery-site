# Garden Recovery — что подменить в прототипе на финальные ассеты

Прототип `index.html` собран как самодостаточный HTML-файл с встроенным CSS и интерактивом. Палитра, шрифты и общая стилистика взяты с действующего `gardenrecovery.me`. Ниже — что именно нужно подменить перед публикацией и где это в коде.

---

## 1. Логотип

**Сейчас в прототипе:** текстовый логотип «Garden Recovery / Medical Residence» в шрифте Cormorant Garamond.

**Что подменить:** реальный графический логотип с действующего сайта.
URL исходника на действующем сайте: `https://gardenrecovery.me/wp-content/themes/gardenrecovery/assets/img/hero-logo.png`

**Где в коде.** Поиск по `class="logo"` — два места: шапка и футер. Заменить блок:

```html
<a href="#" class="logo">
  <span class="logo-mark">Garden Recovery</span>
  <span class="logo-sub">Medical Residence</span>
</a>
```

на:

```html
<a href="#" class="logo">
  <img src="path/to/logo.svg" alt="Garden Recovery" style="height:64px;width:auto">
</a>
```

Лучший формат — SVG. Если только PNG — запросить у владельца сайта оригинальную векторную версию.

## 3. Шрифты

**Сейчас подключены через Google Fonts:**
- **Manrope** — основной sans-serif для текста и заголовков (шрифт действующего сайта).
- **Cormorant Garamond** — серифный для логотипа-заглушки.

Когда подменишь логотип на изображение, Cormorant можно убрать из `<link>` и `--serif` в CSS — он не будет использоваться больше нигде. Это сэкономит загрузку.

Если у бренда есть свой кастомный шрифт — добавить через `@font-face` в `<style>` и подставить в переменные `--sans` / `--serif`.

---

## 4. Логотип в шапке/футере (уточнения)

В прототипе использован **двухстрочный текстовый логотип** — как на действующем сайте (там тоже строчное «Garden Recovery» крупно и под ним «MEDICAL RESIDENCE» small caps). Если на финальном логотипе компании эти строки уже включены в готовый PNG/SVG — заменяй блок целиком на `<img>`. Если это две отдельные строки в брендбуке — оставь текстовую версию, но подмени шрифт на брендовый.

## 5. Hero-видео

**Сейчас в прототипе:** декоративные blob-формы (CSS), без видео. Под текстом УТП — пустое пространство справа с цветовыми пятнами в стиле Адриатики.

**Что добавить:** hero-видео фоном.

В коде ищи `<section class="hero" id="hero">` и замени блок `<div class="hero-blobs">` на:

```html
<video class="hero-video" autoplay muted loop playsinline poster="path/to/hero-poster.jpg">
  <source src="path/to/hero-video.mp4" type="video/mp4">
  <source src="path/to/hero-video.webm" type="video/webm">
</video>
```

И добавь в `<style>` правила:

```css
.hero-video{
  position:absolute;inset:0;width:100%;height:100%;
  object-fit:cover;z-index:0;opacity:.45;
}
.hero::after{
  content:'';position:absolute;inset:0;
  background:linear-gradient(180deg, rgba(240,235,230,0) 30%, rgba(240,235,230,.8) 100%);
  z-index:1;
}
.hero-content{z-index:2;position:relative}
```

Спецификация видео — см. `03_materialy_na_razrabotku.md`, п. 2.1.

## 6. Фотографии команды

**Сейчас в прототипе:** круглые placeholder-кружки с инициалами вместо фотографий.

**Что подменить:** реальные портреты с действующего сайта.

В коде ищи `<div class="member-photo">` (9 мест). Заменить на:

```html
<div class="member-photo">
  <img src="path/to/photo.jpg" alt="Имя" style="width:100%;height:100%;border-radius:50%;object-fit:cover">
</div>
```

Список соответствий — в `04_kopirayt.md`, раздел 6 «Команда».

## 7. Видео команды

**Сейчас в прототипе:** кнопки «Смотреть видео» — без действия (заглушки).

**Что добавить:** при клике открывать модалку с плеером и видео-файлом конкретного специалиста.

В коде ищи `<button class="member-watch">` (9 мест). При финальной интеграции:
- Передавать в карточку `data-video="path/to/video.mp4"`.
- На клик — открывать `<dialog>` или модалку с `<video controls autoplay>`.
- Подключать `.vtt` субтитры RU + EN.

## 8. Контактные каналы

**Сейчас в прототипе:** в блоке «Контакты» три ссылки-заглушки (`#`).

**Что подменить:** реальные ссылки на действующие каналы.

В коде ищи `<div class="contact-channels">` и подмени:
- `href="https://wa.me/..."` — WhatsApp.
- `href="https://t.me/..."` — Telegram.
- `href="mailto:hello@gardenrecovery.me"` — реальный e-mail (если другой).

Все три используются на действующем сайте — забрать с `gardenrecovery.me`.

## 10. Отправка формы — интеграция CRM

**Сейчас в прототипе:** форма консультации валидирует поля локально, выводит toast «Получили…», в console.log выводит данные. Не отправляет.

**Что добавить:** отправку на CRM (Bitrix / amoCRM / другое) или на e-mail через формовый бэкенд (Formspree / Getform / собственный).

В коде ищи `form.addEventListener('submit', ...)` и замени блок `console.log(...)` на `fetch(...)` запрос к выбранному backend-у.

Поля, которые передаются:
- `name` — имя
- `contact` — телефон / username
- `channel` — WhatsApp / Telegram / Звонок
- `program` — какая программа интересует (предзаполняется при клике «Обсудить формат» в карточке программы)

## 13. Open Graph / Twitter Card

**Сейчас в прототипе:** не настроены.

**Что добавить** в `<head>`:

```html
<meta property="og:type" content="website">
<meta property="og:title" content="Garden Recovery — медицинская резиденция на Адриатике">
<meta property="og:description" content="Приватная резиденция для восстановления психоэмоционального состояния...">
<meta property="og:image" content="https://gardenrecovery.me/og-image.jpg">
<meta property="og:url" content="https://gardenrecovery.me">
<meta name="twitter:card" content="summary_large_image">
```

Картинка `og-image.jpg` — 1200×630 px, в эстетике резиденции, без текста или с минимальным текстом.

## 15. Адаптация под мультиязык RU + EN

**Сейчас в прототипе:** только RU. Кнопка переключения языка показывает заглушку «EN version coming soon».

**Что сделать:**
- Дублировать структуру в `/en/index.html` или использовать i18n через JSON-словари + JS.
- Переключатель языка в навигации обновить — пусть переходит на нужный URL.
- Добавить в `<head>`:

```html
<link rel="alternate" hreflang="ru" href="https://gardenrecovery.me/ru/">
<link rel="alternate" hreflang="en" href="https://gardenrecovery.me/en/">
<link rel="alternate" hreflang="x-default" href="https://gardenrecovery.me/ru/">
```
